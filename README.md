# free-survey-form
Solution for Build a Survey Form

This is a simple survey form built with HTML and CSS. It includes fields for name, email, age, role, recommendation, favorite features, and comments.

## Files
- `index.html`: The main HTML file containing the survey form.
- `styles.css`: The CSS file for styling the form.

## How to Run
1. Start a local server: `python3 -m http.server 8000`
2. Open `http://localhost:8000` in your browser.
